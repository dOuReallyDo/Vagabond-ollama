# ICDB Travel Assistant — Project Start

## Vision
Build a production-grade Travel Proposal Assistant inspired by Vagabond, fully aligned with:
https://icorallidibeatrice.it/

Primary objective: COMMUNITY BUILDING (no monetization in MVP).

## Style principles (mandatory)
- Sea destinations, small islands, low crowds
- Prefer minimal car dependence
- Avoid generic mass-tourism resorts unless justified
- Authentic accommodations and local food
- Method-driven planning: nothing left to chance
- Explicit seasonality/weather considerations
- Budget transparency: always ranges (min/max) + assumptions
- Evidence-backed: every operational claim has sources or is marked uncertain
- Tone: pragmatic, reassuring, structured, checklist-oriented; no hype

## MVP Done
- Embeddable widget in WordPress
- Server-side proxy for LLM/tools (no API keys in browser)
- Strict JSON output contract + runtime validation
- Evidence links, map links, cost ranges + assumptions
- Rate limiting, basic logging, error handling
- Staging + production deploy

## Non-goals (MVP)
- Login / payments / dynamic pricing perfection
